## Usage

Interacting with the "My Daily Tasks" application is straightforward and intuitive.

*   **Adding a Task**: To add a new task, simply type its description into the input field labeled "Add a new task..." and click the `+` icon next to it. The task will appear at the bottom of your list.

*   **Marking as Complete**: Each task has a circular icon on its left. Click this icon to toggle the task's completion status. A green checkmark indicates a completed task, which will also be styled with a strikethrough and grayed-out text.

*   **Editing a Task**: To modify the text of an existing task, click the `ğŸ“` (pen) icon associated with that task. A prompt will appear, allowing you to enter a new description for the task.

*   **Deleting a Task**: If you wish to remove a task, click the `ğŸ—‘ï¸` (delete left) icon next to it. The task will be instantly removed from your list.

All your changes (adding, editing, deleting, and completing tasks) are automatically saved to your browser's Local Storage, ensuring your to-do list persists across sessions.
